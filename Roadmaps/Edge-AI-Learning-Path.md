# Edge AI Learning Path

This document outlines a learning path for mastering Edge AI techniques.

## Stages
- **Beginner**: Learn Python, TensorFlow, PyTorch basics, and edge computing concepts.
- **Intermediate**: Study model compression (quantization, pruning, distillation).
- **Advanced**: Implement and deploy Edge AI models on IoT devices or edge hardware.

## Resources
- **Books**: 'Deep Learning for IoT' by John Smith, 'Edge AI Fundamentals'.
- **Courses**: Coursera’s Edge AI specialization, Udemy’s IoT and AI courses.
- **Papers**: NeurIPS, CVPR, and IEEE papers on edge computing.

## Milestones
- Build a quantized model for edge deployment.
- Deploy a model on a Raspberry Pi or Jetson Nano.
- Publish a paper on Edge AI optimizations.
